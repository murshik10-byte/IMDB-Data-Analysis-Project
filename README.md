# IMDB-Data-Analysis-ProjectoAn 
interactive Power BI report analyzing movie trends and ratings using the imdb progect.pbix file.

# Tools Used
Power BI: Data modeling and visualization.  

DAX: Custom measures for data analysis.  

JSON Themes: Custom styling via CY26SU04.json

# Key Contents
imdb progect.pbix: The main project file containing the data model and report layout.  

Static Resources: Custom image assets and dashboard branding.

# Quick Start
Install Power BI Desktop.  

Open imdb progect.pbix to view the interactive charts.
